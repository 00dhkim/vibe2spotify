# vibe2spotify

spotify가 한국에서 정식으로 출시되었다!!

근데 나는 그전까지 vibe를 잘 쓰고 있었는데... 음악 리스트를 어떻게 옮기지??

아하, 크롤링해서 자동으로 spotify에 등록해주는 스크립트를 짜 보자!

## steps

1. `vibe_crawl.py` 코드를 통해 vibe에서 곡명과 가수명을 크롤링하여 파일에 담았다.
2. spotify에 추가를 하는 방법으로, `https://open.spotify.com/search/곡명+가수명`으로 검색하는 방식을 택하려고 하였다. 하지만 예상과 달리, (있는 음원인데도) 검색이 되지 않는 경우가 자주 발생하였다.
3. 그 원인으로, 가수명의 경우 `한국명(영문명)`의 방식으로 된 경우 검색이 되지 않는 경우를 발견하였다. 예를들어, `celebrity 아이유`라고 검색하면 결과가 잘 나오지만 `celebrity 아이유(IU)`라고 검색하면 결과가 나오지 않았다. 문제는 내가 크롤링한 데이터는 `한국명(영문명)` 형식의 가수명을 가진 값들이 여럿 있었다.
